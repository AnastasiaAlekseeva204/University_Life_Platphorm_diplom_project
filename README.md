# University Life Platform 🎓

**University Life Platform** — это веб-платформа, разработанная в рамках дипломного проекта, предназначенная для цифровизации и упрощения взаимодействия студентов, преподавателей и администрации внутри университета.

## 📌 Основные возможности

* **Личный кабинет:** Профиль пользователя с информацией об обучении.
* **Система уведомлений:** Оповещения о важных событиях и сообществах.
* **Форум/Чат:** Пространство для коммуникации между студентами и преподавателями.(возможно в будущем)
* **Активность студентов:** Показывает насколько студент заинтересован в мероприятиях, сообществах и насколько сильно он участвует в них.


## 🛠 Стек технологий

* **Backend:** Python 3.x, Django
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** PostgreSQL / SQLite
* **Инструменты:** Git, Virtualenv

## 🚀 Установка и запуск

Чтобы запустить проект локально, следуйте этим шагам:

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/AnastasiaAlekseeva204/University_Life_Platphorm_diplom_project.git
cd University_Life_Platphorm_diplom_project

```


2. **Создайте и активируйте виртуальное окружение:**
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

```


3. **Установите зависимости:**
```bash
pip install -r requirements.txt

```


4. **Примените миграции:**
```bash
python manage.py migrate

```


5. **Запустите сервер:**
```bash
python manage.py runserver

```


Проект будет доступен по адресу `http://127.0.0.1:8000/`.

## 📂 Структура проекта

* `/myproject` — основной модуль, где хранится все.
* `/views` — модуль управления авторизацией, профилем, принятия участия в  мероприятий или добавление в  сообщества, также вывод всех мероприятий и сообществ.
* `/models` — хранит базу данных.
* `/static` & `/templates` & `/staticfiles` — фронтенд составляющая.
